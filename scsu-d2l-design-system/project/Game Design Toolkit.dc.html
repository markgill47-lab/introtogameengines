Static diagrams and game-art paste straight into the D2L HTML editor. The live widgets (dialogue, inventory, sliders, HUD, collision demo) rely on scripts — embed them via an HTML file upload or iframe, or use them as static illustrations where D2L strips JavaScript. Every color, label, and shape is editable.
