# The Input System

> **Covered:** Week 12 · **Organ of:** [[common-anatomy|the Common Anatomy]]

Devices in, actions out. Every engine has modernized this layer recently, and Week 12 explains why that happened everywhere at once.

Traces of that modernization show up in the engine profiles: [[unity|Unity]] ships **two input systems** (the churn problem), and [[unreal|Unreal]]'s **Enhanced Input** ships half of Week 12 in the box. In [[threejs|Three.js]] you are the input system — browser events in, your own action mapping out.

A classic LLM trap lives here too: the model will confidently hand you Unity's legacy input in a new Input System project. See [[llm-tutor|version drift]].
