# The Scripting Runtime

> **Covered:** Week 3, and then every week forever · **Organ of:** [[common-anatomy|the Common Anatomy]]

Where your code runs, hooked into the loop. C# in [[unity|Unity]], GDScript or C# in [[godot|Godot]], C++ or Blueprints in [[unreal|Unreal]], JavaScript in [[threejs|Three.js]].

The loop it hooks into is the first of the [[what-is-a-game-engine|three promises]]: read input, update everything, draw the frame, and you write the "update everything" part. Week 3 is entirely about this loop, and it's a better week than that sentence makes it sound.

Your language choice here is question two of [[how-to-choose|How to Choose]]: you'll write a lot of it, so pick where you want to live.
