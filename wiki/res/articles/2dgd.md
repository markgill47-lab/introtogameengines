# 2D Game Development: From Zero To Hero

> **Author:** community compendium · **Read it:** [free PDF/EPub via GitHub](https://github.com/2DGD-F0TH/2DGD_F0TH) · **License:** CC BY-NC-SA

A 500+ page community compendium, engine-agnostic, with code in multiple languages. Covers loops, collision, algorithms, and general design.

## Where it pairs with the course

A useful second angle on loops, collision, and algorithms, especially for students in code-first engines ([[w01:threejs|Three.js]] and [[w01:honorable-mentions|MonoGame]] people, this means you). It's 2D-focused and community-authored, so depth varies chapter to chapter: treat it as a reference to dip into, not a text to read through.

One license note for the curious: the ShareAlike license permits adaptation, unlike [[nystrom|Nystrom's]], if excerpt-and-modify ever mattered to you.
